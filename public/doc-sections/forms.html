Forms

Basic Structure

Forms in Formula are designed to be as easy and semantic as possible. Inside a <form> tag. Your inputs will go inside tags with field classes usually <label class="field"> or <div class="field">. Place your submit button at the bottom inside a <div class="actions">.

Layout Classes

  • stacked arranges <label> tags above <input> tags
  • single makes the submit button inline with the form
$formsStacked: true; in your config will stack all forms.

Responsive Forms

By default forms will be 100% wide and fill their parent elements and will become stacked forms at smaller screen sizes. Set $formsRespondTo: false; in your configuration to disable this and use the responsive class on individual forms.

<form>
  <!-- Field -->
  <label class="field">
    <span>Email</span>
    <div class="input">
      <input type="Email">
    </div>
  </label>

  <!-- Actions -->
  <div class="actions">
    <input type="submit" value="Login">
  </div>
</form>

Text Fields

Simple inputs are wrapped with a <label class="field"> element. Place your label text inside a <span> tag . Your <input> elements should be wrapped inside <div class="input">. You should use this syntax for inputs such as <input type="text">, <textarea> and <select>

<label class="field">
  <span>Label</span>
  <div class="input">
    <input type="text">
  </div>
</label>

Radio Buttons and Checkboxes

For <input type="radio"> and <input type="checkbox"> inputs, apply the input class to a <ul> and place your <input>s wrapped inside <label> tags in the <li> tags of the list. This creates a nice semantic list of options.

Options 
<div class="field">
  <span>Options</span>
  <ul class="input">
    <li><label><input type="checkbox" checked>One</label></li>
    <li><label><input type="checkbox" checked>Two</label></li>
  </ul>
</div>

Fieldsets and Grouped Fields

<fieldset> elements can be used to group fields togather. Create the standard <fieldset> and <legend> elements. to get started.

Once you have a field set you can group fields on the same line as children of a <div class="fields two|three|four"> element. You can group 2-4 fields on the same line.

Inside your fields elements you can put multupile <div class="field"> elements with a <input> and <div class="label">

You can also place <div class="field"> tags inside your <fieldset> to create inputs that span the whole width of the <fieldset>

Profile 
<fieldset>
  <legend>Profile</legend>
  <div class="two fields">
    <div class="field">
      <input type="text">
      <label for="">First</label>
    </div>
    <div class="field">
      <input type="text">
      <label for="">Last</label>
    </div>
  </div>
  <div class="field">
    <input type="email">
    <label for="">Email</label>
  </div>
</fieldset>
Location 
<fieldset>
  <legend>Location</legend>
  <div class="three fields">
    <div class="field">
      <input type=text>
      <label for="">City</label>
    </div>
    <div class="field">
      <select>
        <option>Oregon</option>
        <option>Washington</option>
        <option>California</option>
      </select>
      <label for="">State</label>
    </div>
    <div class="field">
      <input type=text>
      <label for="">Zip</label>
    </div>
  </div>
</fieldset>
